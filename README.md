# 🦖 Dino Endless Runner Game

A fun and interactive endless runner game built with **HTML, CSS, and JavaScript** inspired by the Chrome Dinosaur game.

👉 **Live Demo:** https://squirrelk6755-ctrl.github.io/dino-endless-game/

---

## 🎯 How to Play

- Press the **SPACE key** to make the dinosaur jump.
- Avoid incoming cactus obstacles.
- The score increases over time.
- The game speeds up gradually.

---

## 🛠 Built With

- 🔹 HTML
- 🔹 CSS
- 🔹 Vanilla JavaScript
